Escape
======

This is a rewrite of the well-known game Escape, in C++/Qt/OpenGL.

Dependencies
------------

You'll need classical build tools (`build-essentials`), and the
following libraries:

* Qt (>=4)
* OpenGl (>= 2)

Building
-------

```
$ qmake
$ make clean
$ make -j4
```

Running
-------

```
$ ./escape
```
